# diabetes-risk-prediction
## Final Integrated Capstone Project

The final capstone project combined all prior milestone work into a complete end-to-end machine learning analysis focused on diabetes risk prediction using BRFSS health survey data.

The integrated project included:
- A technical report for a professional data science audience
- A non-technical executive summary for business stakeholders
- Model comparison and evaluation
- Exploratory data analysis
- Ensemble learning and clustering methods
- Business and healthcare recommendations
---

## Dataset
- Source: BRFSS 2015 Health Survey Dataset
- 250,000+ health survey responses
- Includes demographic, behavioral, and health-related indicators

Key features include:
- BMI
- Age
- Blood Pressure
- Cholesterol
- Physical Activity
- General Health

---

## Machine Learning Models
- Logistic Regression
- Random Forest
- Gradient Boosting
- K-Nearest Neighbors (KNN)
- K-Means Clustering
- DBSCAN

---

## Key Results
- Gradient Boosting achieved the strongest overall classification performance
- BMI and age were among the strongest predictors of diabetes risk
- Ensemble learning methods outperformed baseline linear models
- Clustering techniques identified distinct health behavior patterns

---

## Project Documentation

### Milestone 1 — Foundations & Overfitting Analysis
Focused on Logistic Regression, Random Forests, feature scaling, regularization, and overfitting mitigation techniques.

### Milestone 2 — Advanced Modeling & Clustering
Expanded analysis using KNN, Gradient Boosting, K-Means clustering, DBSCAN, PCA visualization, and hyperparameter tuning.

### Final Technical Report
Comprehensive end-to-end machine learning analysis including preprocessing, EDA, ROC comparison, feature importance analysis, and healthcare implications.

### Executive Summary Report
Non-technical business-focused report summarizing major findings, model performance, and healthcare recommendations.
